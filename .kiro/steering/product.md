# Product Overview

atl-fetch は、HTTPリクエストを行うためのNode.js CLIツールおよびライブラリです。

## Core Capabilities

- **CLI インターフェース**: コマンドラインからHTTPリクエストを実行
- **ライブラリAPI**: プログラマティックにHTTPリクエストを構築・実行
- **npmパッケージ**: 公開レジストリから簡単にインストール可能

## Target Use Cases

- スクリプトやCI/CDパイプラインでのHTTPリクエスト実行
- 他のNode.jsプロジェクトへの組み込みライブラリとしての利用
- 開発者向けのHTTPデバッグ・テストツール

## Value Proposition

- シンプルで直感的なAPIデザイン
- TypeScriptネイティブで型安全
- 軽量で依存関係が最小限
